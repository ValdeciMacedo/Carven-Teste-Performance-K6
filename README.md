# 🚀 Carven - Testes de Performance com k6

Projeto de testes de performance desenvolvido para avaliar o comportamento do site:
👉 https://carven.com.br/

---

## 🎯 Objetivo

Garantir estabilidade, performance e escalabilidade da aplicação sob diferentes cenários de carga.

---

## 🧪 Tipos de Testes

- Smoke Test → valida disponibilidade
- Load Test → simula carga normal
- Stress Test → identifica limites do sistema
- Spike Test → simula picos de acesso

---

## 🛠️ Tecnologias

- k6
- JavaScript
- Docker
- GitHub Actions
